# AI Data Analyst

An intelligent AI-powered data analytics platform that enables users to upload datasets and interact with them using natural language. The application leverages Large Language Models (LLMs) to generate insights, answer business questions, create visualizations, and produce analytical reports without requiring SQL or programming knowledge.

## Features

* Upload CSV datasets
* Natural language data querying
* Automated data analysis
* AI-generated business insights
* Interactive visualizations and charts
* PDF report generation
* User-friendly Streamlit interface
* Powered by Groq LLM

## Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* LangChain
* Groq LLM
* ReportLab

## Project Workflow

1. Upload a dataset
2. Explore data automatically
3. Ask questions in natural language
4. Generate insights and visualizations
5. Download analytical reports

## Installation

```bash
git clone https://github.com/harshsingh-commits/AI-Data-Analyst-agent-.git
cd AI-Data-Analyst-agent-
pip install -r requirements.txt
streamlit run app.py
```

## Environment Variables

Create a `.env` file and add:

```env
GROQ_API_KEY=your_groq_api_key
```
